Instructions

Laissez SVP un espace entre chaque 2 opérandes
N'oubliez pas l'opérateur = à la fin
La liste des opérations n'est pas complète
Les opérations possibles :

= : égal

+ : addition

- : soustraction

* : multiplication

/ : division entière

% : modulo

^ : exposant

a : nombre aléatoire
